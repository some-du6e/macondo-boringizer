## How to install

1. Get the [latest release](https://github.com/some-du6e/macondo-boringizer/releases/latest)
2. Unzip the file
3. Go to `chrome://extensions/` and turn on developer mode
4. Click `Load unpacked` and select the unzipped file
5. Done

## Contributing
Requirements: 
- bun 1.2.0
- nodejs 22.12+ (just in case)
1. Install packages
```bash
bun i
```
### running it in dev
```bash
bun run dev
```

### building for chrome/firefox/edge
```bash
bun run build:chrome/firefox/edge
```

## Features

- Gets rid of the macondo "farm" and replaces it with boring buttons and cards 
- Good enough compatability with [Macondo Utils](https://chromewebstore.google.com/detail/macondo-utils/ncdoiigcdhkcgeebfbhkfijglhjckhd)
- Option to grayscale the entire page
- Option to lock the shop
- Option to preload the shop
